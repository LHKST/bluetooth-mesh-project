TCP RPi server that works as a gateway between a TCP client and a BLE connected device. When running tcpserver it should take a port number as an argument. The TCP client should initiate connection by entering: IPADRESS PORTNUMBER In a network utility (such as telnet or netcat), where IPADRESS is the IP adress of the server; PORTNUMBER is the portnumber that the server has opened a socket at.
